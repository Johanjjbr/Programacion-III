## Instalacion sencilla

1. Copiar la carpeta `parcial_maximiliano_faria` dentro de `htdocs` si se usa XAMPP.
2. Importar en MySQL el archivo `database/parcial_n2_2026.sql`.
3. Revisar los datos de conexion en `includes/conexion.php`.
4. Abrir `login.html` en el navegador.
